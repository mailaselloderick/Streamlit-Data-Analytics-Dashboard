
# 📊 Streamlit Data Analytics Dashboard

A professional interactive data analytics dashboard built with Python and Streamlit.

---

## Features

- 📊 Interactive Dashboard
- 📈 Plotly Charts
- 📉 Matplotlib Charts
- 🌍 World Map
- 📅 Date Filter
- 🔍 Search Dataset
- 📊 Summary Statistics
- 📋 Missing Values Analysis
- 📦 Correlation Matrix
- 📈 Histograms
- 📊 Scatter Plots
- 📥 Download Dataset
- 🌙 Dark Mode
- 📱 Responsive Layout

---

## Technologies

- Python
- Streamlit
- Pandas
- Plotly
- Matplotlib

---

## Installation

Clone the repository

```bash
git clone https://github.com/mailaselloderick/Streamlit-Data-Analytics-Dashboard.git
```

Install packages

```bash
pip install -r requirements.txt
```

Run

```bash
streamlit run app.py
```
